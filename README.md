Data Removal Assistant

Проект помогает пользователю сформировать запрос на удаление персональных данных в соответствии с ФЗ-152.

Функционал:
- генерация официального письма
- сохранение запросов
- управление запросами

Стек:
Java, Spring Boot, PostgreSQL

Идея:
сделать удобный инструмент, который помогает людям защищать свои персональные данные и отправлять официальные запросы в компании.
